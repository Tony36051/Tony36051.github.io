发表于 2020-05-23 更新于 2022-04-13 分类于 开发

业务先行，先抄一个业务功能的设计。

功能

商品模块

  1. 商品添加
  2. 规格设置
  3. 上下架

    界面描述

    A. 商品采用列表分页展示，列表上方有搜索功能。
    列表头：商品id，图片，名称，分类，价格，销量，库存，排序，上下架状态，操作列（编辑，删除，拼团）
    B. 商品新增、规格设置
    名称、单位、排序、销量、重量、缩略图、多图

    表结构设计

    product_brand
  • id 自增，pk
  • name 品牌名称
  • logo_url 品牌logo
  • sort_id 排序，默认999
  • status 0：停用；1：启用

product_category

  • id 自增，pk
  • name
  • pid 父分类id
  • sort_id 排序，默认999
  • status 0：停用；1：启用
